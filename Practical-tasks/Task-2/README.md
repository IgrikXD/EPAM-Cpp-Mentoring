# hello_world_FLTK application

## Download and install FLTK package:
For Arch-based systems:
```
sudo pacman -S fltk
```

## How to build and execute?
Create and go to the **build** directiory:
```
mkdir build
cd build
```
Build a project with CMake tool:
```
cmake ..
make
```
Run the application:
```
./hello_world_FLTK
```
